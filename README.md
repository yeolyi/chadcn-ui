shadcn/ui의 드롭인 대체품입니다. 같은 API, 같은 모양, 잊을 수 없는 UX.
모든 컴포넌트는 shadcn 원본과 시각적으로 동일하며, 독특한 인터랙션으로 당신의 사이트를 무시할 수 없게 만듭니다.
