## Cursor

Tailwind v4 [switched](https://tailwindcss.com/docs/upgrade-guide#buttons-use-the-default-cursor)
the default from `cursor: pointer` to `cursor: default` for the button component.

To keep the pointer cursor, add the following to your CSS file:

<Code lang="css" title="globals.css" code={`@layer base {
  button:not(:disabled),
  [role="button"]:not(:disabled) {
    cursor: pointer;
  }
}`} />

### Size

Use the `size` prop to change the button size.

<ExamplePreview>
  <ButtonSize slug={props.slug} client:load slot="preview" />
  <Code slot="code" lang="tsx" code={`import { Button } from "@chadcn/ui/button/${props.slug}"

    <div className="flex flex-wrap items-center gap-4">
      <Button size="xs">Extra Small</Button>
      <Button size="xs" variant="outline"><ArrowUpRightIcon /></Button>
      <Button size="sm">Small</Button>
      <Button size="sm" variant="outline"><ArrowUpRightIcon /></Button>
      <Button size="default">Default</Button>
      <Button size="default" variant="outline"><ArrowUpRightIcon /></Button>
      <Button size="lg">Large</Button>
      <Button size="lg" variant="outline"><ArrowUpRightIcon /></Button>
    </div>
  )
}`} />
</ExamplePreview>

### As Child

Use the `asChild` prop on `<Button />` to make another component look like a button. Here's an example of a link that looks like a button.

<ExamplePreview>

    <Button asChild>
      <a href="/login">Login</a>
    </Button>
  )
}`} />
</ExamplePreview>
