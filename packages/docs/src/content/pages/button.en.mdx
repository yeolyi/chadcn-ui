## Cursor

Tailwind v4 [switched](https://tailwindcss.com/docs/upgrade-guide#buttons-use-the-default-cursor)
the default from `cursor: pointer` to `cursor: default` for the button component.

To keep the pointer cursor, add the following to your CSS file:

<CodeBlock lang="css" title="globals.css" code={buttonSnippets(props.slug).cursor} />
