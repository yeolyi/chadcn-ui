## 커서

Tailwind v4는 버튼 컴포넌트의 기본값을 `cursor: pointer`에서 `cursor: default`로
[변경했습니다](https://tailwindcss.com/docs/upgrade-guide#buttons-use-the-default-cursor).

포인터 커서를 유지하려면 CSS 파일에 다음 코드를 추가하세요.

<CodeBlock lang="css" title="globals.css" code={`@layer base {
  button:not(:disabled),
  [role="button"]:not(:disabled) {
    cursor: pointer;
  }
}`} />

### 자식 요소로

`<Button />`의 `asChild` prop을 사용하면 다른 컴포넌트를 버튼처럼 보이게 할 수 있습니다. 아래는 버튼처럼 보이는 링크 예시입니다.

<ExamplePreview>
  <ButtonAsChild slug={props.slug} client:load slot="preview" />
  <CodeBlock slot="code" lang="tsx" code={`import { Button } from "@chadcn/ui/button/${props.slug}"

export function ButtonAsChild() {
  return (
    <Button asChild>
      <a href="/login">Login</a>
    </Button>
  )
}`} />
</ExamplePreview>
